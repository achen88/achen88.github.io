Break the Social Media Bubble Webapp

Link to GitHub repository

I made a webapp that aims to break the social media bubble by connecting users who want to talk one-on-one, civilly, with others with differing opinions. Upon account creation, users pick preferences about issues, and the server will automatically match users on different sides of issues. I use Node.js, Express, Handlebars, Passport.js (authentication), Socket.io (web sockets), and MongoDB to run my webapp.
