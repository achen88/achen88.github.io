Break the Social Media Bubble Webapp

Link to GitHub repository

I made a webapp that aims to break the social media bubble by connecting users who want to talk to the other side. Upon account creation, users pick preferences about issues, and the server will match them with those of differing opinions. I use Node.js, Express, Handlebars, Passport.js (authentication), Socket.io (web sockets), and MongoDB.
